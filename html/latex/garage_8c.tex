\doxysubsection{Detailed Description}
Implements the core garage logic including entry, exit, correction, and occupancy tracking. 

This file handles the primary operations of the Parking Garage Management System, such as registering vehicles, logging exits, calculating fees, correcting timestamps, and displaying garage occupancy.
